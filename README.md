# QueueMaster Telegram Bot

Telegram-бот для управления очередью задач в квартире (готовка, вынос мусора, принос воды).

## Требования

- Docker
- Docker Compose

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd queue-master-bot
```

2. Создайте файл .env и заполните его:
```
BOT_TOKEN=your_bot_token_here
MONGO_URI=mongodb://mongodb:27017/
```

3. Запустите бота через Docker Compose:
```bash
docker-compose up -d
```

## Использование

Команды бота:
- `/start` - Начать работу с ботом
- `/тамақ` - Показать кто готовит сегодня
- `/су` - Показать кто должен принести воду
- `/қоқыс` - Показать кто должен вынести мусор
- `/кезек` - Показать все текущие очереди
- `/кесте` - Показать расписание готовки
- `/көмек` - Показать справку

Кнопки:
- "💧 Су әкелдім ✅" - Отметить выполнение задачи по воде
- "🗑 Қоқыс шығардым ✅" - Отметить выполнение задачи по мусору
- "🍳 Бүгін тамақ кім?" - Показать кто готовит сегодня
- "📋 Кезек тізімі" - Показать список очередей
- "📅 Тамақ кестесі" - Показать расписание готовки

## Особенности
- Автоматическая смена очереди по готовке каждый день в 00:00
- Напоминание о готовке в 16:00
- Уведомления в группу при выполнении задач
- Работает только с авторизованными пользователями
- Хранение данных в MongoDB

## Обновление бота

Для обновления бота:
```bash
git pull
docker-compose up -d --build
```
