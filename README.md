# QueueMaster Telegram Bot

Telegram-бот для управления очередью задач в квартире (готовка, вынос мусора, принос воды).

## Требования

- Python 3.11
- MongoDB
- PM2 (для production)

## Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd queue-master-bot
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Создайте файл .env и заполните его:
```
BOT_TOKEN=your_bot_token_here
MONGO_URI=mongodb://localhost:27017/
```

4. Добавьте ID пользователей в файл config.py:
```python
AUTHORIZED_USERS = [
    123456789,  # Пример ID пользователя
    # Добавьте остальные ID
]
```

## Запуск

### Локальный запуск
```bash
python main.py
```

### Production запуск (DigitalOcean)
1. Установите PM2:
```bash
npm install pm2 -g
```

2. Создайте файл конфигурации для PM2:
```bash
pm2 ecosystem create
```

3. Запустите бот через PM2:
```bash
pm2 start main.py --name queue-master-bot
```

## Использование

Команды бота:
- `/start` - Начать работу с ботом
- `/еда` - Показать кто готовит сегодня
- `/вода` - Показать кто должен принести воду
- `/мусор` - Показать кто должен вынести мусор
- `/статус` - Показать все текущие очереди
- `/помощь` - Показать справку

Кнопки:
- "Я принёс воду" - Отметить выполнение задачи по воде
- "Я вынес мусор" - Отметить выполнение задачи по мусору

## Особенности
- Автоматическая смена очереди по готовке каждый день в 00:00
- Смена очереди по воде и мусору после нажатия соответствующей кнопки
- Работает только с авторизованными пользователями
- Хранение данных в MongoDB
